What is This?

A simple tool to keep track of your Medications.

Considering this the MVP for a much larger project I'm working on.

The idea here is to be able to create a list of your medications, and add them to that list,

delete them from that list, and even edit them in that list. Later, I will add

login/account functionality, so that multiple users can use the app. Also, a calendar that will

show the days that meds have been taken effectively, and change color(green/red) depending on user activation.

These elemnts will segue into a larger project, which is intended to be a sort of personal assistant type thing. Still working out the kinks, really.
